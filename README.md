# psm

Pypi Source Manager: fast switch between different Pypi Source: pypi, double, aliyun.


This package is inspired by [Pana/nrm](https://github.com/Pana/nrm).

## Installation

```
pip install psm
```


## Usage

### Unix

##### list all pypi source

```
psm ls
```

##### change pypi source

```
psm use douban
```

##### show current source

```
psm show
```
### Windows 

##### list all pypi source

```
python -m psm ls
```

##### change pypi source

```
python -m psm use douban
```

##### show current source

```
python -m psm show
```
